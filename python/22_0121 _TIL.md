- 재귀함수는 꼭 그림을 그려보기(네모박스 겹쳐서)

```python
# 회문 판별 함수
def is_pal_recursive(word):
    if len(word) <= 1:
        return True
    if word[0] == word[-1]:
        return is_pal_recursive(word[1:-1])   
    else:
        return False
# 재귀함수는 자기 자신을 호출하는 함수라서 return을 무조건 자기를 반환하도록 생각없이 짜지 말자.
# 자기 자신을 반환하지 않도록 짜면 값이 결과로 나올 수 없다.
```
